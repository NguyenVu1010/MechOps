# RobotOps

Robot Operations Platform — hạ tầng vận hành robot, PLC và Industrial IoT.

- Bắt đầu: đọc `constitution.md` → `CLAUDE.md` → `docs/product/README` 
- Trạng thái: `docs/test-status.md` (49 test, tick tự động kèm evidence)
- Contract: `specs/` (AsyncAPI 3 + JSON Schema) — phần open source cùng `agent/`, `probe/`, `protocol/`
- Lệnh: `make verify` · `make gen` · `make status` · `make hw-test ID=... TESTER=...`

⚠️ Module path đang là `github.com/robotops-io/...` — đổi thành org thật của bạn TRƯỚC commit đầu tiên (grep toàn repo).
