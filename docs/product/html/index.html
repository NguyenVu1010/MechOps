Lớp hạ tầng vận hành robot, PLC và Industrial IoT. Không điều khiển robot — vận hành robot. Một trang tổng hợp: chiến lược → kiến trúc → roadmap → milestone → định nghĩa "xong".

M2 · Dashboard + ros2-probetuần 7–10

Device list/detail, RBAC · probe: node status + AMR profile · bắt đầu tiếp cận design partner

battery + node status robot thật trên dashboard
M3 · OTA v1 ⭐tuần 11–16

Registry, manifest, health check, staged rollout · auto-rollback tại agent · test rút điện giữa chừng

deploy version lỗi → tự về version cũ, không can thiệp

rút mạng → cảnh báo ≤ 1 phút
M5 · Terminal + White-label + Installertuần 21–26

PTY + xterm.js + audit log · theming per-instance · install.sh idempotent

người ngoài team cài instance mới < 30 phút
