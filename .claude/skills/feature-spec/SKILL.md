# Feature Spec (vòng ngoài)
1. Tạo specs/features/<mX-tên>/ theo TEMPLATE.md: spec.md → clarify.md → tasks.md.
2. clarify.md: liệt kê MỌI điểm mờ thành câu hỏi, DỪNG chờ founder trả lời — không tự giả định.
3. tasks.md: mỗi task ≤ nửa ngày, bắt buộc gắn ≥1 test ID từ docs/product/05-test-catalog.md. Task không có ID = task không hợp lệ.
4. Xong vòng ngoài mới vào vòng trong (CLAUDE.md).
